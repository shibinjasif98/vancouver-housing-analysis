# Vancouver Housing Analysis

A data analytics project analyzing Vancouver housing affordability using public datasets.

## Project goals
- Compare rent and income trends
- Identify affordability by area (if neighborhood data is available)
- Build a dashboard and a short report with insights

## Tech stack
- Python (pandas, numpy, matplotlib/seaborn)
- Jupyter notebooks
- Power BI / Tableau (later)
- Git + GitHub

## Folder structure
- data/raw: original datasets (not tracked)
- data/processed: cleaned datasets
- notebooks: exploration and analysis notebooks
- src: reusable scripts (cleaning, feature engineering)
- reports: written findings
- dashboards: Power BI/Tableau files

## Project Scope

This project analyzes housing affordability in Vancouver, Canada by comparing
rental price trends with income growth from 2019 to 2024.

Geographic Scope:
- Vancouver Census Metropolitan Area (CMA)

Key Question:
- Has rental cost growth in Vancouver outpaced income growth?

Primary Metric:
- Rent-to-income ratio (affordability benchmark at 30%)
